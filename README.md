## 📌 О проекте

Единый раздел, объединяющий весь кэшбэк, акции и бонусы пользователя в одном месте.
Показывает аналитику, прогноз выгоды, персональные офферы и ИИ-советы по максимизации кэшбэка.

## ⚙️ Стек

|---|---|
| Backend | FastAPI + pandas |
| Frontend | React + TypeScript + Tailwind CSS |
| Графики | Recharts |

| Деплой | Railway |

## 🚀 Запуск локально

```bash
# Через Docker
docker-compose up --build

# Backend вручную
cd backend
pip install -r requirements.txt
uvicorn app.main:app --reload

# Frontend вручную
cd frontend
npm install
npm run dev
```

## 🧩 Фичи и гипотезы

### Сводка балансов
Три карточки с рублями, милями и баллами Браво — первое что видит пользователь.
**Гипотеза:** пользователь должен сразу понять свою ценность от банка, без лишних кликов.

### График истории кэшбэка
Линейный график по месяцам за последний год.
**Гипотеза:** визуальный рост мотивирует продолжать пользоваться картой.

### Прогноз выгоды
Расчёт на основе среднего кэшбэка за последние 3 месяца.
**Гипотеза:** конкретные цифры ("накопите ещё 12 000 миль к декабрю") убеждают лучше абстрактных обещаний.

### Персональные офферы партнёров
Фильтрация по финансовому сегменту (LOW / MEDIUM / HIGH).
**Гипотеза:** нерелевантные офферы раздражают, поэтому каждый сегмент видит своё.

### Геймификация
Стрики и бейджи на основе реальной истории начислений.
**Гипотеза:** визуальный прогресс и достижения создают привычку заходить в приложение.

### ИИ-советник
Персональный совет на основе профиля и истории пользователя.
**Гипотеза:** конкретный совет ("вы тратите на авиа мало, но у вас All Airlines") повышает вовлечённость.
